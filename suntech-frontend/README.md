**Required:**

**npm installation**
  * https://www.npmjs.com/package/npm
**http-server installation**
  * https://www.npmjs.com/package/http-server
**bower installation**
  * https://www.npmjs.com/package/bower

**How to run:**

in the root folder (/suntech/), execute:


* npm install
* bower install
* http-server -p 8090
* Open http://localhost:8090
